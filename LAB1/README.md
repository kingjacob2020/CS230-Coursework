# Lab 1 - HTML structure

This lab introduces the basics of HTML document structure and semantic markup.

## Features Implemented
 - Valid HTML document structure
 - Multi-page navigation
 - Lists and tables
 - Image with accessibility alt text
 - Contact form with labels and textarea
 - Demonstration of block vs inline elements

 ## Files
 - `index.html` - Main webpage
 - `contact.html` - Contact page with form
 - `images/` - Contains images used in the project

 ## Skills Practiced
 - Writing semantic HTML
 - Understanding webpage structure
 - Creating forms and navigation
 - Accessibility basics
